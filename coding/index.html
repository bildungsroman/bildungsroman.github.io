Coding

JS and serverless all the things!

My main stack consists of JavaScript frameworks: React, Gatsby, and NodeJS. I've worked with Python and Golang in the past, but have forgotten much of the former and wasn't a huge fan of the latter.

I've worked on serverless applications since 2018, primarily using AWS services such as Lambda, API/HTTP Gateway, DynamoDB, SQS, SES, and S3. I'm very familiar with serverless architecture and have worked on both the frontend and backend of serverless applications. If you don't believe me, ask Jeremy Daly - I was named a Serverless Star ⭐ in his Off-by-none newsletter. I'm an AWS Certified Cloud Practitioner (which, TBH, I only got to get access to the certification lounge at re:Invent 🤷‍♀️☕). One day I hope AWS makes a serverless developer certification, which I will eagerly get.
